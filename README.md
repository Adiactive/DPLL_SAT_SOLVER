# DPLL_SAT_SOLVER
A simple SAT solver based on [DPLL algorithm](https://en.wikipedia.org/wiki/DPLL_algorithm)

## Build

- Clone this repository

```
$ git clone https://github.com/Adiactive/DPLL_SAT_SOLVER.git
$ cd DPLL_SAT_SOLVER
```

- Compile using cmake

```
$ mkdir build
$ cd build
$ cmake ../
$ make
```
